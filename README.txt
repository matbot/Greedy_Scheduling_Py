-In order to run the program, activity.py:
    1. Make sure the program file is located in a folder with the act.txt file containing the standardized input.
    2. To run activity.py: $ ./activity.py
    3. Results will be printed to the terminal according to program specifications.

Input file act.txt contains integers with the structure:
number_of_activities
activity_number start_time finish_team (for each activity in number_of_activities)
(may repeat until EOF)
